# job-leyu-lab

## 项目简介

`job-leyu-lab` 是一个用于归档和发布独立 HTML 页面的仓库。  
仓库内包含多个独立的网页文件，每个页面均为完整、可独立运行的 HTML 文档。  
本仓库不针对任何特定域名或网站，仅作为技术内容存档与展示使用。

## 目录结构

```
job-leyu-lab/
├── pages/           # 存放各独立 HTML 页面
├── assets/          # 页面引用的资源文件（样式、脚本、图片等）
└── README.md        # 本说明文件
```

## 页面归档说明

- 每个 HTML 页面均为独立项目，彼此之间没有依赖关系。
- 页面可直接在浏览器中打开运行，无需额外服务器环境（部分页面可能依赖网络资源）。
- 页面内容可能包含实验性功能、技术演示或学习笔记，不代表任何正式产品或服务。
- 页面归档按提交时间顺序存放，不设置特定分类或标签。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 进入 `pages/` 目录，双击任意 `.html` 文件即可在浏览器中查看。
3. 如需修改或二次开发，建议使用代码编辑器打开对应文件。
